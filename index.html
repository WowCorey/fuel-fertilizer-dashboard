Australia's fuel and fertiliser data, without estimates.

Data policy

No fabricated numbers

If a value cannot be verified from a named public source, the relevant dashboard renders a placeholder instead of an estimate. Manual values should only be entered after checking the publisher document listed in the source registry.

Source rights

Publisher terms still apply

Project code, prose and metadata have their own licences. Upstream source data remains under each publisher's rights and citation requirements.
