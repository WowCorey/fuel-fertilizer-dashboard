Public source dashboard

Australia's fuel, food and resilience data, without estimates.

This static site collects source-linked dashboard pages for liquid fuel, food and farm inputs, water pressure, oil production, national fuel security, national fuel status, resource value, state petroleum ledger, strategic resources, defence posture and retail-price breakdowns. Values appear only when they flow through a verified JSON envelope; unavailable values remain visible as placeholders. The National Fuel Security page is the direct fuel-dashboard prototype; the Food, farms & water security page extends the same public-source method into food-system resilience.

Data policy

No fabricated numbers

If a value cannot be verified from a named public source, the relevant dashboard renders a placeholder instead of an estimate. Manual values should only be entered after checking the publisher document listed in the source registry.

Source rights

Publisher terms still apply

Project code, prose and metadata have their own licences. Upstream source data remains under each publisher's rights and citation requirements.
