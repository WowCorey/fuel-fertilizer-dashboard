Unified confidence legend

Status labels are part of the evidence

Unavailable and source-gated data is not a dashboard failure. It is evidence of a public-data gap.

Verified

Source-backed and current enough for its cadence.

Partial

Source-backed, but incomplete by geography, product, timing or concept.

Stale

Source-backed, but outside expected cadence.

Manual

Hand-keyed from a named public source.

Derived

Calculated or selected from a named source envelope.

Unavailable

No public source-safe feed is loaded.

Source-gated

Waiting for verified source, field, period, unit and reuse rights.

Roadmap

Planned dashboard area, not yet populated.

Data policy

No fabricated numbers

If a value cannot be verified from a named public source, the relevant dashboard renders a placeholder instead of an estimate. Manual values should only be entered after checking the publisher document listed in the source registry.

Source rights

Publisher terms still apply

Project code, prose and metadata have their own licences. Upstream source data remains under each publisher's rights and citation requirements.

Next data priorities

Roadmap gaps stay visible until source-safe

The next data work is fuel strategy and policy releases, Queensland fuel-sovereignty follow-up, live station and terminal operations, food/farm/water planning feeds, housing pressure, AI workforce exposure and Brisbane 2032 readiness.

Source gates

No estimate-first roadmap

Warship procurement watch items, forward contracts, state-owned land, regional fuel risk, farm diesel and water-allocation feeds remain source-gated until official/public sources expose exact fields, dates, units and reuse rights.
